       - Read path.join(LINBO_DIR, `start.conf.${id}`) as utf8
       - Call parseStartConf(content) and return result
       - On ENOENT return null. Re-throw other errors.

    3. listNativeGrubConfigs()
       - readdir(GRUB_DIR, { withFileTypes: true })
       - Filter: e.isFile() && e.name.endsWith('.cfg')
       - Map to { filename: e.name, id: e.name === 'grub.cfg' ? 'grub' : e.name.replace(/\.cfg$/, ''), path: path.join(GRUB_DIR, e.name) }
       - On ENOENT return []. Re-throw other errors.

    4. readGrubConfig(id)
       - Resolve filename: id === 'grub' ? 'grub.cfg' : `${id}.cfg`
       - Read path.join(GRUB_DIR, filename) as utf8
       - Return { id, filename, content }
       - On ENOENT return null. Re-throw other errors.

    5. writeGrubConfig(id, content)
       - Resolve filename: id === 'grub' ? 'grub.cfg' : `${id}.cfg`
       - await fsp.mkdir(GRUB_DIR, { recursive: true })
       - await atomicWrite(path.join(GRUB_DIR, filename), content)
       - Return { id, filename, written: true }

    6. reloadLinboServices()
       - const errors = []
       - Try: await execFileAsync('sudo', ['/bin/systemctl', 'reload', 'rsync'])

         IMPORTANT comment: '// tftpd-hpa: must restart — no SIGHUP reload support (reads config only at startup)'
